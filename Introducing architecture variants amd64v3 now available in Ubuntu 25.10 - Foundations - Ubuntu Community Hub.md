# Introducing architecture variants: amd64v3 now available in Ubuntu 25.10 - Foundations - Ubuntu Community Hub

URL: https://discourse.ubuntu.com/t/introducing-architecture-variants-amd64v3-now-available-in-ubuntu-25-10/71312

## Summary

Ubuntu 25.10 introduces opt-in amd64v3 (x86-64-v3) architecture variants, rebuilding a subset of packages for newer CPUs to optionally gain performance while preserving compatibility with older hardware, with a full amd64v3 rebuild planned for 26.04 LTS.
