# Grad

Der Grad eines Knotens gibt an wie viele Kanten mit ihm verbunden sind.

minimum degree:
$$\delta(G)$$
maximum degree:
$$\Delta(G)$$
average degree:
$$d(G)$$
$G$ is $k$-regular if $d(v)=k$ for all nodes $v$, so if all vertecis have the same degree.
