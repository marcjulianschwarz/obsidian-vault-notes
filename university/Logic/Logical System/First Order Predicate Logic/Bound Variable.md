# Bound Variable

A variable $X$ is called bound in a formula $A$ iff it occurs in a subformula
$$\forall X.B$$ of $A$.

So $X$ is bound when it is being used with the [[Universal Quantifier]]. It is called a [[Free Variable]] otherwise.
