# Adjazenzmatrix

A matrix that shows all connections between nodes.

## Advantage

- Edges do not cross

## Techniques

- If [[Graph]] has weights they can be visualized using a [[Color Maps]].
- Order rows based on
  - node name
  - weight ([[Graph Centrality]])
  - group ([[Community Structure]])
